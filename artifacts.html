Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide.

Perfil do recurso MedicationKnowledge - Recurso base informações de medicamento

Perfil do recurso MedicationKnowledge para os ingredientes dos produtos, indicações, dosagem, forma farmacêutica, via de administração, e outras informações relevantes para o uso seguro e eficaz dos medicamentos.

Terminology: Value Sets

These define sets of codes used by systems conforming to this implementation guide.

Dosage Type ValueSet

ValueSet for types of dosage instructions and constraints.
Forma Farmacêutica ValueSet

ValueSet para formas farmacêuticas baseado no INFARMED.
Tipos de Preço INFARMED

ValueSet com os tipos de preço ativos definidos pelo INFARMED.
Via de Administração ValueSet

ValueSet para vias de administração baseado no INFARMED.

Terminology: Code Systems

These define new code systems used by systems conforming to this implementation guide.

Dosage Type CodeSystem

Defines the semantic type of dosage instructions or constraints.
Forma Farmacêutica

CodeSystem para formas farmacêuticas baseado na tabela fornecida
Tipo de Preço INFARMED

CodeSystem para os tipos de preço de medicamentos, baseado na tabela fornecida, incluindo apenas os códigos ativos.
Via de Administração

CodeSystem para vias de administração baseado na tabela fornecida
